# Agent Animation Options

## Goal

Create a review reel with multiple high-quality visual directions for the "launch agents" moment before integrating one into storyboard 03.

- Black dotted Data Harbor background.
- Product-video language: restrained, technical, premium.
- Agents should feel like autonomous AI workers searching sources, not generic bubbles.
- Use luminous SVG lines, moving packets, source cards, and schema/evidence chips.
- Keep each option readable at a glance.
