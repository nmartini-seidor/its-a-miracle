# Storyboard 02

## Goal

Second 10-second storyboard slice.

## Visual Direction

- Same black dotted background as storyboard 01.
- Spanish visible text only.
- Headline first, then four metric/icon items, then issue cards on the right.
- Metric area should feel table-like but have no visible borders.
- Numbers continue changing until the storyboard ends.
- Add a hidden-cost message below the metrics, supported by translucent dollar signs popping behind the table.
